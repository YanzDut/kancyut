Latihan Box Model CSS

Properti yang digunakan:

  • margin: Jarak di luar elemen
  • padding: Jarak di dalam elemen (antara konten dan border)
  • border: Garis tepi elemen
  • width & height: Lebar dan tinggi elemen

Menggunakan * { box-sizing: border-box; } agar padding dan border termasuk dalam perhitungan width/height

Ini adalah contoh kotak teks dengan border dan padding. Padding 20px memberikan ruang di dalam kotak, sedangkan border 2px solid #3498db memberikan garis tepi berwarna biru.

Keterangan:

Dengan box-sizing: border-box, total width dan height elemen sudah termasuk padding dan border, sehingga lebih mudah dalam mengatur layout.
